# More Training

**The Elite Performance Coaching Platform**

## Overview
More Training is a comprehensive coaching platform connecting elite coaches with athletes worldwide. Built with React + Vite, it features AI-powered training plan generation, real-time communication, and seamless integration with Garmin/Strava.
