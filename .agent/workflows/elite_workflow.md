# Elite Performance Workflow

Ce workflow assure que chaque ajout de fonctionnalité ou modification de code est validé par l'expertise du **Elite Performance Architect**.

## Étapes du Workflow

1.  **Analyse de l'Intention Sportive**
    *   Avant de coder, demandez-vous : "Comment cette fonctionnalité aide-t-elle l'athlète à mieux s'entraîner ou à mieux récupérer ?"
## ÉTAPES DE TRAVAIL

### PHASE 1 : DÉVELOPPEMENT & VALIDATION
1.  **Analyse & Développement Focalisé**
    *   Implémentez la logique technique (SOLID, Type Safety) et le design premium (Rich Aesthetics).
    *   Utilisez des clés de traduction temporaires ou la langue du projet pour la rapidité.
    *   Vérifiez la performance (re-renders, accessibilité).

2.  **Validation Utilisateur**
    *   Présentez la fonctionnalité pour validation.
    *   Attendez le "GO" de l'utilisateur sur le design et le fonctionnement.

### PHASE 2 : FINALISATION GLOBALE (Post-Validation)
3.  **Expansion Multilingue Systématique**
    *   **CRITIQUE :** Traduisez toutes les nouvelles clés dans `src/core/services/translations.ts` pour ES, EN, FR, IT, DE, CA.
    *   Vérifiez les termes techniques sportifs spécifiques à chaque langue.

4.  **Optimisation Visibilité (SEO & GEO)**
    *   Générez/Mettez à jour les schémas **JSON-LD** pour la fonctionnalité.
    *   Gérez les balises Meta dynamiques (Title, Description, OG) si nécessaire.
    *   Optimisez pour les crawlers d'IA (structure claire, sémantique HTML5).

5. **Rapport de Performance Final**
    * Générez un résumé des bénéfices "Performance" et de la conformité globale.
